Deklaracja zmiennej - var, let, const

Aby utowrzyć zmienną, inaczej mówiąc zadeklarować ją mamy do wyboru kilka możliwości:

  • var - deklaruje zmienną o zakresie funkcyjnym, przez długie lata najpopularniejszy sposób np:

    var data1 = 3; // pojedyncze = przypisuje wartość do zmiennej

  • let - dodane w ES6 / ES2015 deklaruje zmienną o zakresie blokowym

    let someData = 1;

  • const - dodane w ES6 / ES2015 deklaruje stałą o zakresie blokowym, nie może ulec zmianie

    const dataCantChange = 22;
